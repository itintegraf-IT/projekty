Kam umístit Reporty v navigaci?

Report je CEO/manažerský nástroj — kde by měl žít?

A

Nová záložka v Správě (Admin)

Správa — záložky
Uživatelé Číselníky Presety Audit log Pracovní doba 📊 Reporty
Dashboard obsah se zobrazí zde...

Výhody

  • Konzistentní s existující navigací
  • Přístup mají jen ADMIN a PLANOVAT
  • Žádná nová stránka — jen nová záložka

Nevýhody

  • Správa je dost nabitá (6 záložek)
  • Report je jiný typ obsahu než konfigurace
B

Vlastní stránka /reporty s tlačítkem v hlavním headeru

Header — hlavní navigace
Integraf
Správa Rezervace Expedice 📊 Reporty

Výhody

  • Report je "first-class citizen" — důstojné místo
  • Přístup z hlavní navigace, jeden klik
  • Správa zůstane čistá (jen konfigurace)

Nevýhody

  • Nová stránka = nový route
  • Musí se ošetřit oprávnění (jen ADMIN + PLANOVAT)
