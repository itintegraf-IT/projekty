Jak DTP edituje DATA výrobní sloupec?

Vyber přístup, který ti sedí nejvíc. DTP bude moci: měnit DATA status (dropdown) + zaškrtnout data.ok — ale ne datum.

A — Kontextové menu (pravý klik)

DTP klikne pravým na blok, zobrazí se menu s DATA statusem a OK přepínačem. Stejný vzor jako u planéra (stav zakázky).

B — Zjednodušený edit panel

Klik/dvojklik na blok otevře pravý panel s omezeným pohledem pro DTP: jen DATA status dropdown a OK checkbox. Datum uzamčeno.

C — Popover přímo z chipu

Klik na DATA chip (D 15.5) otevře malý popover s dropdownem a OK checkboxem. Nejrychlejší workflow — vše bez opuštění timeline.

Co bude DTP vždy moct:  změnit DATA status (dropdown — Chybná data / Připraveno / Vysvíceno…) + přepnout data.ok ✓ / ✗   |  Zůstane uzamčeno: datum dodání dat (edituje pouze admin a plánovač)
